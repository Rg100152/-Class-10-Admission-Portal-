# 🎓 Class 10 Admission Portal

> **A Modern, Responsive & Digital School Admission Management System**

The **Class 10 Admission Portal** is a modern web-based admission platform designed to simplify the student admission process. It provides an intuitive interface for students to register, submit applications, upload documents, pay admission fees, and track their admission status online.

This project is built as a frontend prototype using **HTML5, CSS3, and Vanilla JavaScript**, featuring a clean UI, responsive design, and smooth animations.

---

## 📌 Project Overview

The portal digitizes the complete admission workflow, reducing paperwork and making the admission process faster, easier, and more transparent for both students and school administrators.
